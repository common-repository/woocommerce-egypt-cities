== Description ==

A short plugin that adds Egypt cities to WooCommerce's list of states. This can be useful if you're using some kind of shipping methods that requires cities/states to calculate the shipping fees, using woocommerce shipping calculator.

== Installation ==

Before installing this plugin, make sure you've woocommerce plugin already installed and activated.

1. Upload `woocommerce-egypt-cities` to the `/wp-content/plugins/` directory or you can install it directly from wordpress dashboard. Go to `Plugins -> Add New -> search for "Woocommerce Egypt Citites" -> Install`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you're good to go. If you're using any plugins that requires States/Cities for shipping calculation, you can go to the plugin settings and check States/Cities dropdown list and you will find all Egypt citites listed there.

== Screenshots ==
